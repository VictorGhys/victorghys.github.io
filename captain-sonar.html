Captain Sonar

Captain Sonar is a 2D action game. You need to avoid rocks, mines and torpedo's. But you can't look at the sonar and steer the ship at the same time. The goal is to reach the end without hitting anything. The main challenges of this game were making the sonar and displaying it correctly on the hud. The movement of the ship was also challenging to make it feel right, like controlling a big ship. It was also my first time using the 2D part of the unity engine. This game jam made me realise that 56 hours is a very short time to make a game. And I also prefer to team up with artists, so I can focus 100% on the code in the jam.

The game was made for the backtrace game jam in 56 hours in Unity. The theme was "Share knowledge".
