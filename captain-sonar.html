Captain Sonar

Captain Sonar is a 2D action game. You need to avoid rocks, mines and torpedo's. But you can't look at the sonar and steer the ship at the same time. And the goal is to reach the end without hitting anything. The main challenges of this game were making the sonar and displaying it correctly on the hud. The movement of the ship was also pretty challenging to make it feel right. It was also my first time using the 2D part of the unity engine.

The game was made for the backtrace game jam in 56 hours in Unity.
