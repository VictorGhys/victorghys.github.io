Super Mario 64

A clone project of Super Mario 64 for Graphics Programming 2. In this course we learned the basics of 3D rendering with DirectX11. We started with a basic component-based framework that contains some fundamental classes and systems and had to extend this framework with new features every week. These features include adding new shaders, skinned animation, character controllers, shadow mapping, particles, post processing and more.

Made in c++ and DirectX11. The PhysX SDK from Nvidia is used for the physics. FMOD is used for audio.
